# Gameplay Recording Evidence

This directory stores only gameplay recording indexes and analyses for `{{ProductId}}`.

Recording analysis can be saved as product evidence, but it does not automatically update long-term product facts. Promote conclusions into product-profile.md, gameplay-systems.md, or asset-inventory.md only after explicit confirmation.

Recommended per-recording folder:

recordings/
  YYYY-MM-DD-recording-slug/
    recording-analysis.md
    source-notes.md
