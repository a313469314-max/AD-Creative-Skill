# 内容创作方法迁移到广告创意（对外分享版）

## 适合用在什么上面

适合用于游戏广告、短视频广告、信息流素材和创意脚本训练，尤其适合以下问题：

- 产品卖点很多，但不知道从哪个角度切入。
- 脚本像功能介绍，缺少情绪和故事。
- 开头能吸引点击，但中后段承接弱。
- 创意团队只盯竞品广告，角度越来越同质化。
- 想把短剧、直播、梗图、电影、热点内容里的有效结构迁移到广告里。

## 一句话核心

广告创意本质上是高压缩内容创作：在极短时间内完成筛选用户、抓注意力、调动情绪、建立幻想、给出行动理由。

## 广告创意的完整任务

一条效果广告不只是告诉用户“产品有什么”，而是要完成这条链路：

```text
筛选用户
-> 抓住注意力
-> 调动情绪
-> 建立产品幻想
-> 给出可理解的行动理由
-> 引导下载、注册、付费或继续观看
```

所以做创意时不能只问：

```text
这个产品有什么卖点？
```

还要问：

- 用户为什么会停下来？
- 用户被什么情绪击中？
- 用户什么时候理解产品幻想？
- 用户为什么相信下载后能获得这种体验？
- 素材筛出来的是目标用户，还是只被噱头吸引的低质量流量？

## 三步工作法

内容创作可以抽象成三步：获取信息、找角度、创作。

迁移到广告创意后，对应为：

```text
获取信息：产品、用户、竞品、素材、热点、文化、平台机制
找角度：同一个卖点的反差切口、陌生化表达和情绪入口
创作：脚本、镜头、BGM、节奏、字幕、音效和 CTA
```

最关键的是找角度。信息是原料，创作是加工，角度决定用户有没有兴趣看下去。

## 角度原则

### 情理之中，预料之外

好角度不是胡乱反转，而是让用户感觉：

```text
我没想到
但好像确实应该这样
```

广告中的应用方式：

- 同样是孵化，不写“孵化一个宠物”，而写“最后一个没人要的蛋，开出了最强单位”。
- 同样是养成，不写“升级角色”，而写“Day 1 被追着跑，Day 30 回来秒掉整座战场”。
- 同样是公会战，不写“加入公会战斗”，而写“你以为只是收集角色，直到两个公会为一个稀有单位开战”。

### 陌生化

陌生化不是创造新系统，而是用新视角重新呈现熟悉卖点。

可用操作：

- 换观察者：玩家、敌人、宠物、角色、主播、官方、失败者。
- 换时间点：召唤前一刻、战败后、领奖前、弃坑前、Day 365。
- 换表达场景：发布会、道歉、直播、审判、采访、战地记录、实验记录。
- 换问题：不是“这个系统有什么”，而是“它解决了用户什么情绪”。

## 情绪原则

### 先找情绪把柄

用户不是被信息打动，而是被自己的欲望、焦虑、委屈、好奇、炫耀欲和掌控欲打动。

广告创意要先找情绪把柄：

```text
用户怕什么？
用户缺什么？
用户羡慕什么？
用户想赢过谁？
用户想拥有什么？
用户想逃离什么？
用户想证明什么？
```

游戏广告常见情绪把柄：

- 弱小变强。
- 普通变稀有。
- 被压制后反杀。
- 低成本获得高价值。
- 选择正确带来巨大回报。
- 收集完整带来占有感。
- 角色、宠物、军团或资产归我所有。
- 现实受限，游戏里获得力量和身份。

### 爽感可以粗糙，但必须准确

短内容里，用户不一定要求逻辑完整，但要求情绪准确。广告里不要过度解释系统机制，而要先兑现情绪：

```text
先让用户看到结果出现
先让用户看到体型变大
先让用户看到角色登场
先让用户看到敌人被击败
先让用户看到奖励到账
```

机制解释只能服务爽感，不能拖慢爽感。

## 节奏原则

### 高密度推进

广告创意不应机械按秒数切段。真正的结构是阶段任务：

```text
钩子段：抓注意力，并形成一个让用户想继续看的问题
承接桥：把钩子的注意力转译到产品资产 / 产品幻想
核心兑现：让用户看到产品满足的主要欲望
玩法证明：证明这不是纯噱头，下载后有对应体验
质量筛选：展示成长、收集、战斗、社交、长期目标等深度信号
行动收束：CTA / 奖励 / 下一步动机
```

秒数只能作为诊断参考，不是创作边界。钩子可以很短，也可以持续更久；只要钩子内部仍在持续制造新信息、新情绪或新悬念，就不算拖。

真正的问题不是“第几秒回产品”，而是承接桥是否清楚，用户是否能理解钩子和产品幻想之间的关系。

### 情绪地图

脚本不是镜头堆叠，而是一条情绪曲线。

推荐广告情绪曲线：

```text
好奇
-> 意外
-> 紧张 / 期待
-> 爽点兑现
-> 拥有欲 / 羡慕
-> 行动理由
```

每个镜头都要回答它承担哪种情绪。如果一个镜头不制造新信息、新情绪或新转化理由，就应该删掉或压缩。

### 三番递进

短剧和喜剧里的“三番”可以迁移到广告：

```text
第一番：建立预期
第二番：加大反差
第三番：爆点兑现
```

重点是每一层都比上一层更强，不是重复展示同一类画面。

## 人设与资产原则

### 把人设反差转成资产反差

内容爆点常来自人设反差。广告里可以转译为资产反差：

- 普通道具 / 稀有道具。
- 小宠物 / 巨型单位。
- 新手角色 / 领主身份。
- 被追杀 / 反杀。
- 低级阵容 / 完整军团。
- 弱势开局 / 高价值奖励。

反差必须可视化。只用字幕说“变强了”不够，画面必须让用户一眼看出差异。

### 工具人可以存在，但必须服务主幻想

广告里的敌人、路人、竞争者、主播、官方角色都可以工具化，但必须服务核心幻想：

- 衬托资产强。
- 衬托主角逆袭。
- 衬托奖励稀有。
- 衬托选择后果。
- 衬托竞争压力。

不能让工具人抢走产品资产。

## 信息输入原则

### 信息源要跨领域

创意不能只看游戏广告。有效信息源包括：

- 竞品广告：结构、钩子、CTA、素材表现。
- 短剧：高密度爽点、反差人设、付费节点。
- 喜剧：三番、反转、节奏。
- 电影：镜头、叙事弧线、情绪铺垫。
- 直播和短视频：平台原生感、真实感、弹幕感。
- 历史和神话：题材包装和视觉符号。
- 日常生活：真实用户痛点和情绪把柄。
- 社会热点和网络梗：低成本理解入口。

跨领域不是照搬题材，而是提炼结构。

### 信息必须被加工成故事

低效表达：

```text
这个游戏有召唤、养成、战斗、公会、活动。
```

有效表达：

```text
你只剩最后一次召唤机会。
选错只能得到普通单位。
选对，今晚就能翻盘公会战。
```

## 创意生成检查表

生成脚本前检查：

- 这个素材的用户情绪把柄是什么？
- 它的角度是否“情理之中，预料之外”？
- 它的钩子段是否制造强注意力？
- 它是否有清楚的承接桥，把钩子转译到产品资产或产品幻想？
- 它是否完成核心幻想兑现，而不是只停留在噱头？
- 它是否有三番递进，而不是重复堆画面？
- 它是否把系统功能变成了故事动作？
- 它是否筛选到了目标用户，而不是只骗点击？
- 它是否有真实产品承接？
- 它是否存在平台审核或品牌风险？

生成分镜前检查：

- 每个镜头承担什么情绪？
- 哪些镜头负责钩子？
- 哪些镜头负责爽点？
- 哪些镜头负责产品证明？
- BGM 的强拍落在哪些画面？
- 字幕是否补充理解，而不是重复画面？
- 是否有可以删掉的解释性镜头？

投放后复盘检查：

- CTR 好，是否说明钩子有效？
- 3 秒完播率好，是否说明开头节奏成立？
- CPI 好，是否说明安装动机成立？
- 首日 ROI / 付费率好，是否说明用户质量可接受？
- 次留 / 长期 LTV 好，是否说明产品承接成立？
- 如果前端好后端差，是不是情绪或包装筛错了用户？

## 与广告方法论的关系

这套方法补充的是内容创作底层能力，尤其是：

- 角度选择。
- 情绪设计。
- 故事压缩。
- 节奏曲线。
- 反差结构。
- 用户筛选。

它不替代产品定位、素材数据口径、平台审核规则和具体产品的美术资产约束。
