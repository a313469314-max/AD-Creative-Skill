# 全文方法论参考

本目录保存用户提供的“创意 skill 方法论和调用工作流”压缩包中的原始 Markdown 全文。它是参考原典，不替代 `../ad-creative-methodology.md` 的执行规则。

## 文件

- `00-routing-workflow.md`：广告创意方法调用流程。
- `01-performance-ad-creative-methodology.md`：效果广告创意方法论总纲。
- `02-performance-ad-creative-pattern-library.md`：效果广告创意方法卡片库。
- `03-content-creation-to-ad-creatives.md`：内容创作方法迁移广告创意。
- `04-excellent-creative-cases-methodology.md`：优秀素材案例分析方法论。
- `05-gameplay-concept-storyboard-prompt-method.md`：游戏概念分镜提示词方法。

## 使用方式

默认执行顺序：

1. 先读 `../ad-creative-methodology.md`，按精简执行版完成 Phase1 判断。
2. 当需要更细的解释、方法选择依据、案例机制或未来 Phase2 设计时，再查阅本目录全文。
3. Phase1 仍然只输出参考拆解和候选故事方向池；即使阅读了 prompt / 分镜方法，也不能提前创建 production storyboard（制作分镜）、prompt、出图内容或 `script-*` 文件夹。

## 优先级

如果内容之间出现冲突，按以下顺序处理：

```text
product-profile/gameplay-systems
> hook-mapping
> asset-inventory
> materials/memory 素材记忆
> competitors 竞品素材模块
> playbooks
> methodology/ad-creative-methodology.md
> methodology/full/*
```

全文方法论可以补充判断细节，但不能覆盖产品事实、产品增强包约束或 Phase1/Phase2 边界。
