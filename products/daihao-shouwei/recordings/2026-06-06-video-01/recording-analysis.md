## 3. 核心玩法体验链路

第一段录屏呈现的是“新手引导下的首关塔防战斗闭环”：

1. 加载进入关卡，建立轻冒险/英雄小队对抗怪物的题材预期。
2. 进入沙漠主题战斗地图，怪物从上方入口沿弯曲道路推进，玩家防线位于道路两侧和下方。
3. 引导箭头推动玩家点击底部按钮或卡牌，进行单位放置、召唤或升级类操作。
4. 防守单位逐步增加，画面中出现蓝色法师/水晶类单位、绿色弓手/植物类单位、橙色火焰类单位、近战小队或召唤物等多种战斗表现。
5. 中段触发“合成满星英雄”反馈，系统以强弹窗、角色气泡和三选一卡牌表达阶段性成长奖励。
6. 后段敌人密度提高，并出现红色战斗提示条，防守单位通过投射物、范围圈和特效持续清怪。
7. 战斗胜利后进入结算页，展示 MVP/输出统计、通关奖励和确认按钮，形成“进入第一关 -> 学习布防 -> 战斗升级 -> 胜利领奖”的闭环。

## 4. 题材与美术分析

- 题材基调：轻奇幻/卡通冒险。加载页以英雄小队奔跑追逐怪物开场，战斗地图是沙漠遗迹、峡谷道路、神庙入口和黄沙地貌。
- 视觉风格：竖屏 2D 卡通塔防，角色和怪物都是小比例 Q 版形态，单位轮廓清楚，适合移动端快速识别。
- 战斗表达：主要通过固定路径、塔位/单位站位、攻击范围圈、投射物、爆炸/火焰、绿色范围效果和红色警示条表达战斗压力。
- UI 风格：顶部显示关卡/波次/生命等战斗状态，左侧有暂停和倍速类按钮，底部有资源数值、召唤/强化入口和卡牌式操作按钮。整体偏强引导、低门槛上手。
- 成长反馈：`frame-016` 的“合成满星英雄”弹窗使用暗背景遮罩、金色横幅、角色对白和“三选一”卡牌，强化“合成成功/奖励选择”的爽感。
