# 真实录屏证据

本目录用于保存《代号守卫》的真实游戏录屏索引和录屏分析。

规则：

- 只放属于《代号守卫》的录屏与分析。
- 不把其他游戏的录屏放进本目录。
- 录屏分析可以作为证据保存，但不等于自动更新长期产品事实。
- 只有明确确认后，才把录屏分析结论同步到 `product-profile.md`、`gameplay-systems.md` 或 `asset-inventory.md`。

推荐单条录屏目录：

```text
recordings/
  YYYY-MM-DD-recording-slug/
    source/
    evidence/
    recording-analysis.md
    source-notes.md
```
